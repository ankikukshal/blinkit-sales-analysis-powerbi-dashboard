# 🛒 Blinkit Sales Analysis Dashboard | Power BI

An interactive **Blinkit Sales Analysis Dashboard** built in **Power BI** to analyze sales performance, outlet distribution, customer preferences, and product-level insights.

This dashboard transforms raw Blinkit grocery sales data into **actionable business intelligence** using KPI cards, trend analysis, category breakdowns, and outlet performance metrics.

---

## 📌 Dashboard Highlights

* 💰 **Total Sales:** $1.20M
* 📦 **Total Items Sold:** 9K
* 📊 **Average Sales:** 140.99
* ⭐ **Average Rating:** 3.92

---

## 🚀 Features

* 📍 **Outlet-wise sales analysis**

  * Tier 1, Tier 2, Tier 3 performance
  * Outlet size comparison (Small, Medium, High)

* 🛍️ **Product-level insights**

  * Item type sales contribution
  * Fat content analysis (Low Fat vs Regular)
  * Item visibility and weight analysis

* 📈 **Time-based trends**

  * Outlet establishment year trend
  * Yearly sales performance growth

* 🎯 **Interactive filters**

  * Outlet location type
  * Outlet size
  * Item type

* 📋 **Detailed data table**

  * Item identifier
  * Fat content
  * Item type
  * Outlet details
  * Sales performance metrics

## 🎯 Business Problem Solved

This dashboard helps analyze:

* Which outlet type generates maximum sales
* Best-performing item categories
* Sales trend by establishment year
* Customer preference for low-fat vs regular products
* Regional outlet performance comparison
* Inventory and sales optimization opportunities

---

## 💡 Key Insights

* **Tier 3 outlets** contribute the highest sales
* **Medium-size outlets** show strong revenue contribution
* **Low-fat products** have higher demand than regular items
* Certain item categories dominate overall sales
* Older established outlets show stable long-term performance
